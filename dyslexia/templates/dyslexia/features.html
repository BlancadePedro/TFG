    Test para la detección de dislexia

    Descubre si tienes dislexia con nuestro test de 32 preguntas diseñado para evaluar habilidades cognitivas y procesos de lectura.

    • Evaluación detallada de los siguientes procesos cognitivos:
      • Habilidades lingüísticas
      • Memoria de trabajo
      • Procesos perceptuales
      • Funciones ejecutivas
    • Proporciona un diagnóstico claro y preciso de si el usuario tiene dislexia o no.
    • Ayuda al usuario a identificar en que ejercicios presenta mayor dificultad.

    Ejercicios prácticos

    Mejora tus habilidades de lectura y comprensión con nuestros ejercicios de tratamiento diseñados para tratar la dislexia. Encontrarás:

    • Ejercicios de reconocimiento de letras y palabras, diseñados para mejorar la capacidad de lectura y escritura del usuario.
    • Ejercicios de audio para mejorar la asociación de sonidos con imágenes y la identificación de sílabas en las palabras.
    • Ejercicios para mejorar la comprensión y retención de información, como la asociación de imágenes y sonidos con palabras y la identificación de patrones visuales.
    • Ejercicios para mejorar la atención y la concentración, como la resolución de rompecabezas y la realización de tareas de ordenación y clasificación.

    Modelo de Machine Learning

    Nuestro modelo de machine learning ha sido entrenado con un conjunto de datos obtenidos de un estudio realizado en colegios públicos españoles.

    • El modelo de machine learning ha sido entrenado con más de 5.000 registros de usuarios disléxicos y no disléxicos.
    • El modelo utiliza un algoritmo de Support Vector Machine (SVM) que se ha demostrado ser muy efectivo en la clasificación de los datos.
    • El modelo tiene una precisión del 94% en la predicción de la dislexia, lo que lo convierte en una herramienta confiable para la evaluación inicial.
    • El modelo sigue mejorando continuamente gracias al sistema de retroalimentación que permite actualizarlo con nuevos datos de los usuarios y, por lo tanto, mejorar su capacidad de predicción.

    Sistema de retro alimentación

    Mejora tu experiencia y ayuda a nuestro modelo de Machine Learning a adaptarse a los nuevos datos con nuestro sistema de retroalimentación personalizado.

    • Permite al modelo adaptarse y mejorar su capacidad para predecir la dislexia con mayor precisión.
    • Los datos que generan son utilizados para mejorar la precisión del modelo en la detección de la dislexia.
    • Permite que los datos generados por el usuario sean incorporados al modelo de Machine Learning.

    Diseño accesible

    Nuestra página web está diseñada para ser accesible para todas las personas, sin que importen sus capacidades físicas o cognitivas.

    • Diseño adaptable para diferentes dispositivos y tamaños de pantalla.
    • Opción de modificar el tamaño y la fuente del texto para adaptarse a las preferencias de cada usuario.
    • Alternativas textuales para todas las imágenes garantizar la accesibilidad a personas con discapacidades visuales.
    • Controles de navegación compatibles con el teclado para facilitar el acceso a la página web a personas con discapacidades motoras.
    • Compatibilidad con tecnologías de apoyo, como lectores de pantalla y dispositivos de entrada alternativos.
