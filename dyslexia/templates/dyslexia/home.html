    Dislexia Detect

    Nuestra página web ofrece una solución integral la dislexia, que incluye:

    • Una prueba que permite determinar con precisión si se tiene dislexia, facilitando la búsqueda del tratamiento adecuado.
    • Un modelo de Machine Learning entrenado para la detección de dislexia, altamente preciso en la predicción de dislexia
    • Un sistema de retroalimentación incorporado al modelo de Machine Learning, obteniendo resultados actualizados en todo momento
    • Ejercicios prácticos y personalizados para tratar la dislexia, cubriendo distintas áreas y habilidades

    Información sobre la página

    ¿Qué ofrecemos?

    Descubre cómo nuestra página web puede ayudarte a tratar la dislexia con nuestros productos y servicios únicos.

    ¿Cómo detectamos la disexia?

    Descubre si tienes dislexia con nuestro test de 32 preguntas diseñado para evaluar multiples procesos cognitivos.

    ¿Se puede tratar la dislexia?

    Mejora tus habilidades de lectura y comprensión con ejercicios prácticos diseñados para tratar y superar la dislexia.

    Preguntas frecuentes

    Nuestro modelo de Machine Learning se ha entrenado y testeado con un conjunto de datos obtenidos de un estudio realizado en colegios públicos españoles. Se ha logrado una precisión del 94% en la predicción de si una persona tiene o no dislexia. Entendemos que cada caso es diferente y que la precisión puede variar en función de varios factores, por ello, nuestro modelo está en constante mejora gracias al sistema de retroalimentación que se le ha incorporado. Será capaz de aprender y adaptarse en función de los nuevos datos que se le proporcionen.

    Los ejercicios que ofrecemos en nuestra página web están diseñados para trabajar distintos procesos cognitivos, como la memoria, el procesamiento visual o la conciencia fonológica. Lo que se pretende es ofrecer ejemplos dinámicos y cercanos de prácticas que pueden realizar los usuario para tratar y superar la dislexia.

    Nos tomamos muy en serio la privacidad y seguridad de los datos de nuestros usuarios. Todos los datos se manejan de acuerdo con las leyes de privacidad aplicables, y se implementan medidas de seguridad técnicas y organizativas para garantizar la protección adecuada de los datos personales. Además, nunca compartimos información de usuario con terceros sin el consentimiento explícito del usuario.

    Hemos trabajado laboriosamente para asegurarnos de que nuestra página web sea accesible para todos los usuarios, incluyendo aquellos con necesidades especiales y discapacidades. Hemos implementado herramientas y tecnologías de accesibilidad, como el soporte de lectores de pantalla, la navegación con teclado y el contraste alto para mejorar la accesibilidad para aquellos con problemas de visión o motrices. Se han estudiado y aplicado a conciencia las pautas especificadas por la WCAG.

    Actualmente la plataforma está diseñada para ser accesible y gratuita para cualquier usuario registrado que quiera realizar el test de dislexia o practicar con los ejercicios de tratamiento. Sin embargo, es posible que en el futuro se agreguen características o funcionalidades adicionales que requieran de un coste adicional para su uso. En ese caso, se informará a los usuarios de antemano.
